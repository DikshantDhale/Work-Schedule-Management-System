# Work-Schedule-Management-System
CLI-based Work Schedule Management System in C for managing employees, teams, weekly work hours, and performance rankings.

## Features

- Add employees with department and weekly working hours
- Create teams with multiple members
- Track weekly working hours for employees
- Generate Top 5 employees based on total hours
- Check team target achievement
- Generate team rankings
- Check employee reward eligibility
- Update or delete employees
- Merge teams
- Delete teams
- Display team information
- Data persistence using file storage

## Concepts Used

- C structures
- Arrays of structures
- Modular functions
- QuickSort algorithm for rankings
- File handling for data persistence
- Menu-driven CLI program

## Compile
gcc WSMsystem.c
## RUN
./a.out
